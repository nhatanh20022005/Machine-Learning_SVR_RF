Profit Prediction (Superstore Dataset)
Project này tập trung vào việc xây dựng hệ thống dự báo lợi nhuận cho chuỗi cửa hàng bán lẻ (Superstore) sử dụng các thuật toán Học máy (Machine Learning). Đặc biệt, project triển khai cả hai hướng: Xây dựng thuật toán từ đầu (Scratch) và Sử dụng thư viện tối ưu (Scikit-learn) để so sánh hiệu năng.

Các tính năng chính
Mô hình hóa đa dạng: Triển khai Random Forest, Support Vector Regression  và mô hình Ensemble .

Thuật toán "Chạy tay": Tự triển khai SVR  và Random Forest mà không dùng thư viện học máy để hiểu sâu bản chất vận hành của thuật toán.

Tối ưu hóa siêu tham số: Sử dụng cơ chế tìm kiếm lưới (Grid Search) tùy chỉnh để xác định bộ tham số tối ưu dựa trên tập Validation.

Ứng dụng Web (Streamlit): Giao diện tương tác cho phép nhập thông số đơn hàng và nhận dự báo lợi nhuận trực tiếp theo thời gian thực.

Tiền xử lý dữ liệu mạnh mẽ: Xử lý nhiễu (Outliers) bằng phương pháp IQR Clipping, mã hóa Label Encoding và chuẩn hóa dữ liệu với StandardScaler.
